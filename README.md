# 麻雀コーチ付き練習ゲーム

初心者向けに、打牌するたびにおすすめ打牌と理由を表示するブラウザ用の麻雀練習ゲームです。

## できること

- 4人麻雀風の進行
- 手牌、ツモ牌、捨て牌、ドラ表示牌の表示
- クリック打牌
- CPU3人の自動打牌
- おすすめ打牌、理由、役の狙い、鳴き判断、嬉しい牌、初心者向け解説の表示
- 孤立牌、対子、順子候補、両面/カンチャン/ペンチャン、ドラ周りを見たおすすめ打牌
- テンパイ時の待ち牌と待ちの良さの説明
- 他家の直近捨て牌に対するチー/ポン判断の説明
- 相手の捨て牌から狙い・警戒度・危険牌・安全牌を推測する相手分析コーチ
- 現物、スジ、壁、ドラ周辺を使った初心者向けの危険度表示
- リーチ、タンヤオ、ピンフ、役牌、ドラ、一発、七対子の学習表示
- 和了時の点数計算の根拠表示
  - 役ごとの翻
  - ドラの説明
  - 符の仮定
  - 親/子、ロン/ツモ別の支払い

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作る
2. このフォルダの中身をアップロードする
3. リポジトリの `Settings` → `Pages` を開く
4. `Deploy from a branch` を選ぶ
5. `main` ブランチ、`/root` を選んで保存する

しばらくすると、以下のようなURLで開けます。

```text
https://ユーザー名.github.io/リポジトリ名/
```

## ファイル構成

```text
index.html
README.md
.nojekyll
```

`index.html`だけでゲーム本体が動きます。
